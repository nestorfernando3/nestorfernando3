# Nestor Fernando De Leon Llanos

Open-source maintainer building Spanish-language tools for education, writing, and accessible learning.

I create public software for teachers, students, and writers, with a focus on practical classroom workflows, pedagogical UX, and contributor-friendly documentation. My projects are maintained in public GitHub repositories with demos, releases, changelogs, and setup instructions.

## Focus Areas

- Real-time classroom tools and collaborative learning software
- Pedagogical writing tools and Markdown-based workflows
- Spanish localization, documentation, and education-first product design
- Open-source web and desktop apps built with React, Tauri, Vite, Supabase, Electron, and GitHub Actions

## Flagship Open-Source Projects

| Project | What it does | Public signals |

| [Paideia](https://github.com/nestorfernando3/paideia) | Real-time classroom toolkit for teachers and students | Public demo, releases, changelog, MIT license, GitHub Actions |
